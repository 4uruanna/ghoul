# Ghoul

## Notes

Safe imports

```js
import { ... } from 'environments/components.ts';
import { ... } from 'environments/icons.ts';
```

Never use directly

```js
import { ... } from 'ghoul-components';
import { ... } from 'local-ghoul-components';

import { ... } from 'ghoul-icons';
import { ... } from 'local-ghoul-icons';
```
